# Bybit Execution Tracker

Консольное приложение для отслеживания execution events с Bybit Linear Futures через WebSocket API v5.

## Запуск

```bash
dotnet run
```

## Восстановление пропущенных событий

Автоматическое переподключение работает, но восстановление пропущенных событий не реализовано. Можно сохранять ID последнего события и через REST API запрашивать пропущенные при переподключении.
